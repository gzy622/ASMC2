#### 任务4：按组件拆分 CSS 文件

##### 现状分析
当前 `styles.css` 为单文件结构，约 1200+ 行，包含以下模块：
- **基础样式**：:root 变量、reset、body
- **布局组件**：header、main、student-card
- **UI 组件**：dropdown、dialog/modal、scorepad、bottom-sheet
- **功能模块**：roster-xxx（花名册）、asg-xxx（作业管理）、trend-xxx（趋势分析）
- **动画**：keyframes
- **响应式**：media queries

##### 拆分方案

```
css/
├── base.css          # 基础变量、reset、body 样式
├── components.css    # 通用组件：header、student-card、dropdown
├── modal.css         # 弹窗系统：dialog、modal-card
├── scorepad.css      # 记分板组件
├── bottom-sheet.css  # 底部面板
├── roster.css        # 花名册模块
├── assignment.css    # 作业管理模块
├── trend.css         # 趋势分析模块
├── animations.css    # 所有 keyframes
├── responsive.css    # 媒体查询
└── index.css         # 入口文件，导入以上所有模块
```

##### 实施步骤

1. 创建 `css/` 目录
2. 按模块提取 CSS 到独立文件
3. 创建 `index.css` 作为入口
4. 更新 `index.html` 引用新的 CSS 入口
5. 验证所有样式正常加载

##### 验收标准

- [ ] CSS 成功拆分为 10+ 个模块文件
- [ ] 每个文件职责单一，不超过 300 行
- [ ] 入口文件正确导入所有模块
